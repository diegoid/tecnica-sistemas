
Git y GitHub son cosas diferentes pero funcionan integradamente.

Git es un software, GitHub.com es un sitio. 

Son dos cosas separadas. Puedo tener uno sin el otro.

### Git

**Git permite**

- **Control de versiones**: Git permite llevar un seguimiento preciso de los cambios realizados en el código a lo largo del tiempo, lo que facilita la recuperación de versiones anteriores y la comparación entre ellas.

- **Trabajo en Equipo**: Git permite a múltiples desarrolladores trabajar en el mismo proyecto de forma simultánea. Y pueden hacerlo sin pisarse entre si (por ejemplo: uno puede cambiar una parte, otro otro y después fusionan los cambios de los dos en una nueva versión).
  

Cuando trabajo en un proyecto, a ese proyecto se lo llama **"repositorio"**.

Ese repositorio no sólo contiene el "estado actual del proyecto" sino los 
cambios que se hicieron en el pasado.

### GitHub

GitHub permite **alojar repositorios en su web**.

Esto tiene algunas ventajas:

- **Copia de seguridad:** El repositorio queda en la nube además de en la pc. 
  
- **Subir a Github es fácil:** Los cambios que hago en el repositorio los puedo subir fácilmente a GitHub usando Git.
  
- **La subida de contenidos a GitHub es inteligente:**  Si tengo 100 archivos e hice cambios en 12, cuando ejecute un comando para subir los cambios del repositorio, Git va a detectar automáticamente que archivos fueron modificados y sólo va a subir esos archivos.
  
- **Acceso Global:** Otras personas pueden acceder a los repositorios y pueden acceder a los últimos cambios que haga. Podemos hacer que el repositiorio sea público o privado (compartido sólo con quienes queremos).
  
- **Trabajo en equipo mas facil:** Si trabajamos en equipo, otros pueden también hacer sus cambios y subirlos a Github (al repositorio) dentro de una rama (una sección) de ellos.
  
- **Podemos Mostrar Nuestros Trabajos:** Tener proyectos públicos, permite
  que podamos mostrarlos a otras personas (podemos incluir nuestros proyectos en nuestro curriculum) y esto es muy bueno a nivel laboral (para presentarnos a un trabajo o para mostrar trabajos hechos si tenemos un emprendimiento).
