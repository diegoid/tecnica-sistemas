
## Descargar Git

1. Ve a:  
    [https://git-scm.com](https://git-scm.com)
    
2. Hacer clic en **Download for Windows**.
    
3. Instalar.
    

Cuando termina la instalación aparecen **dos programas principales**:

- **Git Bash**
    
- **Git GUI**
    
 **Usamos  "Git Bash"** (es el que casi todo el mundo usa).

El ícono suele ser una **ventana negra con un logo verde de Git**.
